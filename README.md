# Markdown Run Book (_mdrb_)

[![deno doc](https://doc.deno.land/badge.svg)](https://doc.deno.land/https/deno.land/x/mdrb)

Execute a markdown file with Deno.

```sh
deno install --unstable -Arfn mdrb https://deno.land/x/mdrb/mod.ts
```
